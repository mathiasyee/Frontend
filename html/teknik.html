Teknik

Grunder i Java

Java är ett objektorienterat programmeringsspråk som används för att bygga allt från små program till stora system. Jag har lärt mig att skapa grundläggande program med klasser, metoder och variabler för att lösa problem effektivt.

Kontrollstrukturer

Med hjälp av if-satser, switch-satser och loopar som for och while, kan jag skapa program som hanterar olika scenarier och itererar genom data för att uppnå specifika mål.

Objektorienterad programmering

Jag har lärt mig grunderna i OOP, inklusive koncept som arv, inkapsling och polymorfism. Detta gör det möjligt att skapa modulära och återanvändbara kodlösningar i Java.

Java och Databaser

Jag har använt JDBC för att ansluta Java-program till databaser som MySQL. Detta inkluderar att skapa, läsa, uppdatera och ta bort data (CRUD-operationer) via SQL-frågor i mina program.

Databashantering

Jag har en förståelse för grunderna i SQL, inklusive skapande av tabeller, relationer och enkla frågor. Jag kan skriva SELECT-, INSERT- och UPDATE-frågor för att hantera data på ett strukturerat sätt. Detta inkluderar att skapa, läsa, uppdatera och ta bort data (CRUD-operationer) via SQL-frågor i mina program.
